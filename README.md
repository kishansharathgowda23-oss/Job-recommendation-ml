# Job Recommendation System
 Top 10 Finalist – AI IN ACTION Hackathon

## Problem Statement
Many students and job seekers struggle to:
- Choose the right career path
- Identify required skills
- Understand skill gaps

Existing platforms like LinkedIn or Naukri do not provide:
- Personalized recommendations
- Skill gap analysis
- Career roadmap

---

##  Solution
This project builds an AI-powered system that:
- Extracts skills from resumes
- Matches them with job datasets
- Recommends best-fit job roles
- Identifies missing skills
- Provides a learning roadmap

---

##  How It Works
1. Upload Resume (PDF)
2. Extract Skills using NLP
3. Convert text using TF-IDF
4. Compare with job dataset (Cosine Similarity)
5. Predict job role using ML models
6. Show:
   - Job Recommendations
   - Skill Gap
   - Career Path
